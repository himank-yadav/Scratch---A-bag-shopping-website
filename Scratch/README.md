# 🛍️ Bag Shopping Website

A full-stack e-commerce web application for shopping stylish bags. The application allows users to browse products, add items to their cart, and securely manage their accounts with authentication. It is built using **Node.js**, **Express.js**, **MongoDB**, and **EJS**.

---

## 🚀 Features

- 🔐 User Authentication (Register/Login/Logout)
- 🔒 Secure password hashing using **bcrypt**
- 🎫 JWT-based Authentication
- 🛒 Add products to cart
- ❌ Remove products from cart
- 📦 Browse all available bags
- 💰 Automatic price calculation with discounts
- 📱 Responsive UI using **Tailwind CSS**
- 💬 Flash messages for user actions
- 🗄️ MongoDB database integration with Mongoose
